# Fake News Detection using DSA & Machine Learning

## Overview
This project focuses on detecting fake news (rumors) from Twitter threads using 
data structures and machine learning techniques.

## Dataset
- PHEME dataset (charliehebdo, ebola, ferguson, etc.)
- Data is structured as conversation threads (tree-like)

## Approach
- Parse thread into tree structure
- Extract features (text + structure)
- Train classification model

## How to run
pip install -r requirements.txt
python src/model.py
